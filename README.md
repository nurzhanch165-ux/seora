# SEORA — интернет-магазин корейской косметики и здоровья

Современный магазин на **Next.js 14 + TypeScript + Tailwind CSS** с собственной дизайн-системой и нарисованными вручную SVG-иконками. Реализует ключевые сценарии технического задания: каталог из двух разделов, карточки товаров со всеми характеристиками, корзина, регистрация по телефону, формирование заказа в Excel, личный кабинет с загрузкой скриншота оплаты, админ-панель со статусами заказов и общим Excel для склада.

## Запуск

Нужен установленный [Node.js 18.18+](https://nodejs.org).

```bash
npm install
npm run dev
```

Откройте http://localhost:3000

Сборка production-версии:

```bash
npm run build
npm start
```

## Доступы

- **Личный кабинет**: зарегистрируйтесь на `/register` (логин — номер телефона).
- **Админка**: отдельный вход на `/admin/login`. Данные по умолчанию: логин `admin`, пароль `admin123` (задаются в `src/store/adminAuth.ts`).

## Хранение данных (текущая версия)

Для быстрого старта и демонстрации данные клиентов, заказы и избранное хранятся в браузере (`localStorage` через Zustand). Каталог и категории — в файлах `src/data`. Это позволяет запустить сайт без сервера и базы данных.

## Куда развивать (этап 2)

- Перенос данных в БД и серверный API (рекомендуется **Supabase**: PostgreSQL + Auth + Storage для скриншотов).
- Онлайн-оплата (Stripe / PayPal / корейские и локальные платёжные системы).
- Автоматические уведомления (email, WhatsApp, Telegram, SMS).
- Мультиязычность (рус / кор / англ / каз / узб).
- Оптовые клиенты и цены, минимальная сумма заказа, прайс-листы.
- Автоматический расчёт доставки, интеграция с CRM и Google Sheets.

## Структура

```
src/
  app/            маршруты (App Router)
  components/     UI-компоненты и иконки (собственный SVG-набор)
  data/           каталог, категории, бренды, настройки сайта
  lib/            утилиты: формат, типы, генерация Excel
  store/          состояние (корзина, авторизация, заказы, избранное)
```

## Дизайн

- Палитра: тёплая «бумажная» база, графитовый текст, акцент терракота — стиль luxury-минимализма.
- Шрифты: Lora (заголовки, с кириллицей) + Inter (текст).
- Все иконки нарисованы вручную в едином линейном стиле (`src/components/icons.tsx`), без эмодзи и сторонних библиотек.
- Товарные изображения — фирменные градиентные карточки с глифом категории (`ProductVisual`), легко заменяются на реальные фото.
